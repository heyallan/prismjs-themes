PrismJS themes

Installation

CDN Development (redirect to latest)

<link rel="stylesheet" href="https://unpkg.com/prismjs-themes/dist/{{ theme_name }}.min.css"/>

CDN Production (specific release)

<link rel="stylesheet" href="https://unpkg.com/prismjs-themes@2.0.0/dist/{{ theme_name }}.min.css"/>

NPM package

$ npm install --save-dev prismjs-themes

CSS import

@import url("https://unpkg.com/prismjs-themes@2.0.0/dist/{{ theme_name }}.min.css");
